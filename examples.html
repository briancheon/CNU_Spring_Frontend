HTML 태그 사용 규칙 및 예시

오른쪽의 코드를 본인의 프로젝트에 맞게 수정해서 사용해 보세요.

1. 구조와 텍스트 태그

제목과 문단 (h1~h6, p)

<h1>여기는 헤딩 1 (가장 큼)</h1>
<h3>여기는 헤딩 3</h3>
<p>일반적인 문단(Paragraph)을 
작성할 때 사용하는 태그입니다.</p>

여기는 헤딩 1 (가장 큼)

여기는 헤딩 3

일반적인 문단(Paragraph)을 작성할 때 사용하는 태그입니다.

강조하기 (strong, em)

<p>
  이 텍스트는 
  <strong>아주 중요합니다</strong>. 
  이 텍스트는 
  <em>조금 강조합니다</em>.
</p>

이 텍스트는 아주 중요합니다. 이 텍스트는 조금 강조합니다.

2. 링크와 이미지 태그

다른 페이지로 이동하기 (a)

<a href="https://github.com" 
   target="_blank">
  GitHub으로 이동하기
</a>
GitHub으로 이동하기

* target="_blank"를 넣으면 새 창에서 열립니다.

이미지 넣기 (img)

<img 
  src="https://via.placeholder.com/150" 
  alt="프로필 사진 자리표시자"
  width="100"
>
3. 목록 (리스트) 태그

순서가 없는 목록 (ul, li)

<h3>나의 취미</h3>
<ul>
  <li>음악 감상</li>
  <li>코딩 배우기</li>
  <li>독서</li>
</ul>

나의 취미

  • 음악 감상
  • 코딩 배우기
  • 독서

순서가 있는 목록 (ol, li)

<h3>오늘의 할 일</h3>
<ol>
  <li>HTML 복습하기</li>
  <li>프로필 페이지 만들기</li>
  <li>저녁 먹기</li>
</ol>

오늘의 할 일

  1. HTML 복습하기
  2. 프로필 페이지 만들기
  3. 저녁 먹기
4. 연락처 폼 (사용자 입력 받기)

폼 요소들 (form, input, textarea, button)

<form>
  <label for="uname">이름:</label>
  <input type="text" id="uname" placeholder="홍길동">
  <br><br>

  <label for="msg">메시지:</label><br>
  <textarea id="msg" rows="3"></textarea>
  <br><br>

  <button type="submit">보내기</button>
</form>
